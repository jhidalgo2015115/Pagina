BEATBOX

CAMPEONES MUNDIALES
Etimologícamente Beatbox proviene de: beat(sonido) box(caja) lo que en español podria traducirse como "Caja de Sonidos"

El Beatbox es una forma de realizar música solo con sonidos que provengan desde la boca, garganta, cuerdad vocales yhasta incluso la nariz. El Beatbox posee una amplía gama de sonidos que pueden realizarse, gracias a esto los "beatboxers" -como se les denomina a los que practican este arte- no tienen limitantes para generar nuevas rutinas. Se cree comúnmente que el beatbox solo se utiliza para recrear canciones ya exitentes, pero esto no es así, cada beatboxer tiene la capacidad de crear rutinas propias, originales, habiendo incluso beatboxers famosos que han sacado sus propios temas haciendo beatbox.
